# Automate Security Token Operations with the n8n-nodes-polymesh Community Node

We're excited to announce that Velocity BPA has released a new community node for n8n: n8n-nodes-polymesh. This integration brings the power of Polymesh blockchain automation directly into your n8n workflows, enabling seamless security token operations without writing a single line of code.

## The Challenge with Security Token Automation

Security tokens represent a revolutionary step in digital finance, but managing them comes with unique challenges. Traditional blockchain platforms weren't built with regulatory compliance in mind, creating friction for institutions dealing with identity verification, compliance checks, and settlement processes. Polymesh addresses these challenges with a purpose-built blockchain for regulated assets, but integrating it into business workflows has required significant technical expertise—until now.

## What n8n-nodes-polymesh Brings to Your Workflows

This community node enables you to automate critical security token operations directly within n8n, including:

- **Identity Management**: Automate onboarding and verification processes for investors
- **Compliance Automation**: Trigger compliance checks and enforce transfer restrictions programmatically
- **T+0 Settlement**: Execute instant settlement workflows that integrate with your existing business systems
- **Token Operations**: Create, transfer, and manage security tokens through visual workflows

By bringing Polymesh into n8n, you can connect security token operations with your CRM, notification systems, databases, and hundreds of other integrated services—all through an intuitive visual interface.

## Getting Started

Installing the Polymesh node is straightforward. In your n8n instance, simply run:


npm install n8n-nodes-polymesh


After installation, restart n8n, and you'll find the Polymesh node available in your node palette, ready to be dragged into your workflows.
